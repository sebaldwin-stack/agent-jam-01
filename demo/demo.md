## Demo Guide

When recording your demo:

1. **Show the problem first** — What does the agent forget or fail at without this skill?
2. **Install/activate the skill** — Show how easy it is to get started
3. **Demonstrate the solution** — Run the skill and show the agent now remembers / can do X
4. **Wrap up** — Brief recap of value

Keep it 2-5 minutes. Human narration is encouraged but not required.
